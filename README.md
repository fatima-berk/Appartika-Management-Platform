#  Appartika - Plateforme de Gestion Locative Immobilière

**Appartika** est une solution web complète (Full-Stack) conçue pour moderniser et simplifier la gestion des allocations d'appartements.

---

##  Présentation du Projet
L'objectif d'Appartika est de digitaliser les processus manuels de location en offrant une interface fluide pour trois types d'utilisateurs :
* **Locataires :** Recherche, réservation et suivi des paiements.
* **Propriétaires :** Gestion des biens, suivi des revenus et des disponibilités.
* **Administrateurs :** Supervision globale, modération des avis et gestion des comptes.

##  Stack Technique
Ce projet démontre ma maîtrise des technologies modernes de développement web :
* **Frontend :** React.js (Hooks, Axios pour la consommation d'API).
* **Backend :** Laravel (PHP) - Architecture MVC.
* **Authentification :** Laravel Sanctum (Gestion sécurisée par tokens).
* **Base de données :** MySQL.
* **Modélisation :** UML (Diagrammes de classes et de cas d'utilisation).

## Fonctionnalités Clés
* **Système de Réservation :** Gestion en temps réel des disponibilités.
* **Tableau de Bord :** Visualisation des revenus et statistiques pour les propriétaires.
* **Sécurité :** Protection contre les injections SQL et authentification robuste.
* **Gestion des Avis :** Système de notation pour instaurer la confiance entre utilisateurs.


## Installation
1.  **Backend :**
    ```bash
    cd backend
    composer install
    php artisan migrate
    php artisan serve
    ```
2.  **Frontend :**
    ```bash
    cd frontend
    npm install
    npm start
    ```
